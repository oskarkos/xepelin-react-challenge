## Run Locally

Clone the project

```bash
  git clone https://github.com/oskarkos/xepelin-react-challenge
```

Go to the project directory

```bash
  cd xepelin-react-challenge
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
